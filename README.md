# web-scraping
## Prerequisite
- Python 3
## Setup
- Create virtual environment (This is only needed for the first time)
```bash
python3 -m venv venv
```
- Activate virtual environment
```bash
. venv/bin/activate
```
- Install requirements
```bash
pip install -r requirements.txt 
```

## Run
```bash
python sakurazaka46.py
```
